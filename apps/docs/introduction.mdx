## Welcome to SG Cars Trends API

The SG Cars Trends API provides comprehensive access to Singapore's vehicle registration data and Certificate of Entitlement (COE) bidding results. Our unified API service offers real-time insights into Singapore's automotive market trends.

## Key Features

- **Real-time Data**: Access to the latest vehicle registration and COE data
- **Flexible Filtering**: Filter data by month, make, fuel type, and vehicle type
- **Built-in Analytics**: Comparison tools for trend analysis
- **OpenAPI Documentation**: Interactive API explorer with live examples
- **Automated Updates**: Workflow-based data updates with social media integration
- **High Performance**: Optimized for fast response times with Redis caching

## Data Sources

Our API aggregates data from official Singapore government sources:
- **LTA DataMall**: Land Transport Authority's official data portal
- **Vehicle Registration**: Monthly car registration statistics
- **COE Bidding**: Certificate of Entitlement auction results
